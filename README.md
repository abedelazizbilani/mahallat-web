# mahallat (Magasins)

Mahallat est une application web qui donne l’accès aux utilisateurs de parcourir les magasins a l'aide d'une application mobile.
les utilisateur mobile ont l'acces de voir les magasins dans une diametre bien determine sur le plan de google maps, et de parcour le magasin et ses produit, en plus d'evaluer les magasins et ses produits.
d'autre part pour que ces magasins apparaissent sur le plan, ils doivent s'inscrire a cette plaftforme a l'aide d'une portail web, où ils ils soumettent leurs informations et leur emplacement.
afin d'obtenir l'approbation de l'administrateur, le gestionnaire de ce  peut maintenant ajouter ses produits et les gérer.

## Les prérequis sont:

1. Installer Java SE Runtime Environment (version 8 ou superieure) sur le serveur à administrer
2. Installer un base de donne MySQL
3. Installer serveur Tomcat
